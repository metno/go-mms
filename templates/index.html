What is Hello?

The Hello API provides free access to MET Norway's best ever hello data.

The Hello API is primarily for developers who need to develop scripts or applications that need to say hello to MET Norway's employees. Please read about the terms of use for the API, as well as MET's privacy policy statement

Please email helpdesk@met.no if you have questions about Hello, and it will be directed to the appropriate people. We are looking into setting up a discussion forum in the future, as the old email list has been discontinued.
