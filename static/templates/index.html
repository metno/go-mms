What is MMSd?

MMSd API receives, lists and publishes events about products procuced by a production hub at MET Norway

Posting and watching for new events uses NATS. Listing of all available events for the last few days can be done through a REST endpoint. Please read about the terms of use for the API, as well as MET's privacy policy statement

For more information about MMS see the MMS repo on GitHub

Issues with the MMSd software can be reported directly in GitHub

Please email helpdesk@met.no if you have questions about MMSd, and it will be directed to the appropriate people.
